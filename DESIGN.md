# Design System: 日系手帐 · 打字机

Generated by /plan-design-review on 2026-04-19
Project: my-blog

## Design Philosophy

像在精致的日本手帐本上写字。干净的米白纸张、淡淡的横格线、柔和的点缀色。
不是冰冷的 SaaS 产品，是一页可以安心书写的纸面。

参考: Hobonichi 手帐 (巴川纸)、Kokuyo Campus、Traveler's Notebook、
日式文具美学、打字机字体、手帐文化。

关键词: 清水、米白、低饱和、横格线、手写感、留白。

## Typography

```
标题 (中文):  "Zen Maru Gothic", serif
  - 圆润温暖的日系字体，有亲和力
  - fallback: "Hiragino Maru Gothic", "Noto Serif SC", serif

正文 (中文):  "Noto Sans SC", sans-serif
  - 清晰可读，不抢戏
  - fallback: "PingFang SC", "Microsoft YaHei", sans-serif

英文/数字:    "Courier Prime", monospace
  - 打字机风格的核心字体
  - 用于: 日期、阅读时间、分类标签、导航
  - fallback: "Courier New", monospace

代码:        "JetBrains Mono", monospace
  - 代码块专用
```

## Background Texture

页面背景有淡淡的横格线，像手帐本的内页。

```css
/* 横格线背景 - 用于主内容区 */
background-image: repeating-linear-gradient(
  to bottom,
  transparent,
  transparent 31px,
  var(--line) 31px,
  var(--line) 32px
);
/* 行间距 32px，线高 1px，颜色极淡 */
/* 卡片、导航栏等组件不显示横格线，用纯色背景 */
```

## Components

### 导航栏
- 非固定顶部，随页面滚动
- 背景: var(--bg) 纯色，底部 1px dashed var(--border-light)
- Logo 用打字机字体: "◇ my_blog" Courier Prime, var(--navy)
- 导航链接: 纯文字，无图标。hover 时鼠尾草绿下划线
- 右侧: 搜索图标 + 暗/亮模式切换

### 博客文章卡片
- 横向卡片 (左图右文)
- 封面图: 左侧，200x150，圆角6px，saturate 0.9
- 标题: Zen Maru Gothic, 24px, var(--text)
- 摘要: 2行截断, var(--text-secondary)
- 标签: 小圆角药片，打字机字体，鼠尾草绿浅背景
- 日期 + 阅读时间: 打字机字体，小号，var(--text-secondary)
- 底部: 虚线分隔

### 新闻资讯条目
- 横向紧凑布局 (不是卡片)
- 来源名: 小号打字机字体，var(--accent) 鼠尾草绿
- 标题: 正文大小，中等粗细
- AI摘要: 2行截断，var(--text-secondary)
- 时间: "2小时前" 格式，打字机字体
- 左侧: 细竖线装饰 (var(--accent), 2px)

### 文章详情页
- 双栏: 内容 (max-width: 680px) + 目录 (右侧 sticky)
- 目录: 简洁列表，当前章节 var(--accent) 高亮
- 正文: 16px，行高 1.9，段落间距 24px
- 代码块: var(--bg-secondary) 背景，圆角6px，打字机字体
- 引用: 左侧 var(--accent) 竖线 + var(--accent-light) 背景
- 图片: 圆角 6px，带柔和阴影
- 顶部阅读进度条: var(--accent) 鼠尾草绿，2px 高

### 后台管理 (手帐/笔记本风)
- 横格线背景更明显 (与前台区分)
- 侧边栏导航: 简洁文字列表，打字机字体
- 表格: 虚线行分隔
- 按钮: 鼠尾草绿主按钮，米白次按钮
- 标签用灰粉色 (var(--rose)) 区分状态

## Motion

```
过渡: transition: all 200ms ease
hover 效果: 卡片微微上移 2px + 阴影加深
暗色模式切换: 300ms 渐变
无限滚动加载: 淡入 300ms
页面切换: 无动画 (Instant，手帐风格不做花哨动画)
```

## Responsive

```
桌面 (1024px+):  双栏布局
平板 (768-1023px): 新闻侧栏移到下方，单栏
手机 (<768px):  Tab 切换 [博客/资讯]，全宽卡片，目录收起为浮层
```

## AI Slop 排除清单

- NO 紫色/蓝色渐变
- NO 3列特征网格
- NO 圆形图标装饰
- NO 统一大圆角
- NO 装饰性 blob/波浪分隔
- NO emoji 作为设计元素
- NO 彩色左边框卡片
- NO "解锁/赋能/一站式" 等通用文案
- NO 每个区块一样高的模板节奏

### Admin Login (/admin/login)
- 居中卡片, max-width: 400px
- 背景: 淡淡横格线图案 (笔记本纸张感)
- 标题: "📓 my_blog 管理手帐"
- 输入框: 无边框, 底部虚线, 打字机字体 placeholder
- 按钮: 鼠尾草绿 (var(--accent)), 圆角6px, "登 录"

### Admin Dashboard (/admin)
- 手帐/笔记本风格
- 左侧边栏 (240px): 笔记本标签感, 导航列表
- 右侧内容区: 横格纸背景
- 文章列表: 虚线行分隔, 打字机字体日期
- 按钮: 鼠尾草绿主按钮 + 米白次按钮
- 侧边栏项: "○ 文章 ○ RSS源 ○ 统计 ○ 设置 ○ ← 返回首页"

### Admin Article Editor
- Markdown 编辑器 (monospace, 深暖棕背景)
- 右侧实时预览 (使用和前台一样的渲染样式)
- 顶部: 标题输入 + 状态选择 + 发布按钮
- 标签输入: 药片式标签, 回车添加

### Admin RSS Sources (/admin/sources)
- 源列表: 表格, 虚线分隔
- 每行: 名称 | URL | 状态(活跃/停用) | 最后抓取 | 操作
- 添加按钮: 鼠尾草绿 "+ 添加源"
- 状态指示: 活跃=小绿点, 停用=小红点
